HTML est un langage structuré par des balises (tag). Contenu visible dans body. HTML sert a structurer le document. Pour mettre en forme (fonte etc) il faut CSS (feuilles de style) (avant HTML suffisait) Pour rendre dynamique (interactivite, evenements..) il faut JavaScript

On peut faire un titre en HTML avec h1 (titre de niveau 1) mais on ne sait pas exactement ce que ca fait en fct du navigateur

Penser a valider le document dans l'IDE ou sur http://validator.w3c.org
On peut isoler des blocs de contenu avec le tag <section> Utile pour l'accessibilité par exemple, pour les ebooks...etc On pourra avec CSS definir un style par section
On peut aussi creer des sous groupes avec <article> Penser une page comme un ensemble de composants.
On peut definir un titre par section alors qu'avant on ne pouvait en n'avoir qu'un par page (tag h1)

Le tag <a> permet de creer des hyperliens. L'attribut target permet de dire au navigateur d'ouvrir le lien ds un onglet ou pas. lien relatif vers le document a.html lien relatif vers le document a.html lien absolu vers le document a.html lien interne vers un element id="anchor"

Avec le plugin emmet on peut creer du contenu (p*10+tab crée 10 paragraphes vides)

Balises ul

Les balises ul, ol et dl permettent de creer des listes

Exemple

ul

le code EMMET "ul>li*3>{item-$}+tab" va creer la liste suivante

  • item-1
  • item-2
  • item-3

ol

le code EMMET "ol>li*3>{item-$}+tab" va creer la liste suivante

  1. item-1
  2. item-2
  3. item-3

dl

le code EMMET "dl>(dt+dd)*3+tab" va creer la liste suivante

le code EMMET "dl>((dt>{titre item $})+(dd>{definition item $}))*3+tab" va creer la liste suivante

titre item 1
definition item 1
titre item 2
definition item 2
titre item 3
definition item 3

Les tables

le code EMMET "table>tr>td+tab" va creer une table (TR=table row, TD=table division)

col1 col2 col3
foot1 foot2 foot3
cellule 1 cellule 2
cellule 1 cellule 2
cellule 2 cellule 3

Form(ulaires)

Les formulaires permettent de saisir des entrees utilisateurs avec balise <form> et attributs action, type..etc
