Box Model

Chaque element a des dimensions par defaut

Les elmt block (dont le 'display' est block) ont une largeur de 100%: ils prennent toute la place disponible

Les elmt Inline et Inline-block s'etendent et se reduisent verticalement en fct de leur contenu

Les inline n'acceptent pas de changement de dimension, alors que les block et inline-block oui

Il est possible de changer la proprio Display pour passer d'un elmt block et un inline et inversement

On peut modifier les attributs padding(espace entre contenu et bordure) et margin(espace entre elements)

En fct des navigateur, la dimension des elements est calculee differement....(padding+dim contenu, ou dim contenu). Mais on peut utiliser la propriete box-sizing pour choisir comment les calculs de taille sont faits
